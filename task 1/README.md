Task 1. (25 points)
In this task, you get HTML (task1.html), CSS (task1.css), and JavaScript (task1.js) source code of a webpage that lets the user add players with a name, score, and team, then view and manage them in a simple scoreboard. The user can search, filter by team or active status, sort by score, increment scores, activate or deactivate players, and reset all scores. The statistics panel should always correctly show the total number of players, the number of active players, the total score across all players, and the current top-scoring player.

However, the implementation contains multiple (more than 10) bugs that prevent the application from functioning correctly. Your task is to identify, explain, and correct these bugs. Some bugs are visible right away, while the others will require debugging using console.log() messages and trying different inputs. Keep all console.log() you add for debugging in the final submission. You can comment out the lines with bugs and explain the bugs directly in the comments. Then, you add the corrected code and a short explanation of it. Submit the annotated HTML and JavaScript files, including all code you added for debugging.
